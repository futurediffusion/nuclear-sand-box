# Archived Migration Docs

This folder stores phase-by-phase migration artifacts that were previously spread across `docs/` and `docs/architecture/`.

These documents are intentionally preserved for historical traceability, but they are **not** the current architecture source of truth.

Use these docs for:
- change-history reconstruction,
- forensic debugging of why a migration happened,
- reviewing phase acceptance criteria after the fact.

Do **not** use these docs as normative architecture guidance for new changes.

Current authoritative references:
- `docs/architecture/current_sandbox_architecture.md`
- `docs/architecture/ownership/README.md`
- `docs/architecture/sandbox_domain_language_migration.md`
- `docs/architecture/sandbox-contract.md` (process guardrails)
